# Web界面功能更新说明

## 更新内容 (2025-12-11)

本次更新完全按照桌面GUI功能设计，补全了Web界面的拟合和统计功能。

---

## 🎯 新功能一览

### 1. 拟合功能 - 预设模型选择器

**位置**: `/fit` 页面

**新增内容**:
- 拟合模式新增 **"预设模型 (Preset)"** 选项
- 可从10个预定义模型中直接选择:
  - 线性/多项式模型 (M1-M3)
  - 对数模型 (M4, M4B)
  - 反幂级数 (M5, M8)
  - 指数模型 (M7, M7B)
  - 高次衰减 (M6)

**使用方法**:
```
1. 选择拟合模式: "预设模型 (Preset)"
2. 从下拉框选择具体模型 (如: "线性 / Linear")
3. 输入数据后点击"运行拟合"
```

**优势**:
- 无需手动输入模型表达式
- 覆盖最常用的科学计算模型
- 与桌面版完全一致

---

### 2. 统计功能 - 细粒度控制

**位置**: `/stats` 页面

**新增内容**:
- 统计模式简化为两种基础模式:
  - `算术均值`: 普通平均值计算
  - `σ 加权均值`: 根据不确定度加权

- 新增两个独立控制选项:
  - ☑ **使用样本统计 (n-1 校正)**
    - 勾选: 样本统计，适用于估计总体参数
    - 不勾选: 总体统计，适用于描述完整数据集

  - ☑ **使用加权方差计算** (仅加权模式)
    - 勾选: 使用加权平方和计算方差
    - 不勾选: 使用等效点数的非加权方差

**使用场景**:

| 场景 | 统计模式 | 样本统计 | 加权方差 |
|------|---------|---------|---------|
| 测量数据估计真值 | 加权 | ✓ | ✓ |
| 描述测量集合 | 均值 | - | - |
| 加权但简化方差 | 加权 | ✓ | - |
| 总体描述 | 均值 | - | - |

**使用方法**:
```
1. 选择统计模式: "算术均值" 或 "σ 加权均值"
2. 勾选"使用样本统计"(推荐用于实验数据)
3. 如选择加权模式，可进一步配置"使用加权方差计算"
4. 输入数据后点击"计算统计量"
```

## ❓ 常见问题

**Q: 预设模型和多项式拟合有什么区别?**
A: 多项式拟合是指定阶数的多项式，预设模型包含多种类型(对数、指数、反幂等)。预设模型提供了更多样化的选择。

**Q: 什么时候用样本统计?**
A: 当你的数据是总体的一个样本，要估计总体参数时，应使用样本统计(n-1校正)。如果数据就是完整总体，用总体统计。

**Q: 加权方差计算有什么用?**
A: 在加权平均时，如果想要方差也体现权重的影响，就勾选此项。不勾选则使用等效样本数计算简化方差。
