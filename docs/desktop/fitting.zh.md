# 拟合

拟合模块用于对数据进行显式模型拟合，输出参数、不确定度与拟合优度信息，并支持曲线与残差可视化。

## 数据与不确定度

- 选择自变量列与因变量列进行拟合
- 如数据包含不确定度信息，可选择是否按不确定度进行加权拟合
- 当数据中同时存在不同来源的不确定度时，建议结合界面提示选择更合适的加权方式，避免重复计入

## 模型与参数

桌面端支持多种显式拟合模型：

- 多项式模型
- 倒数幂级数
- Padé 与幂律极限模型
- 自定义非线性模型与自洽隐式模型

模型相关参数会随选择动态显示在左侧区域。

## 自定义与自洽隐式模型

自定义模型和自洽隐式模型共用工作台中的公式卡片、参数表和常数表。公式卡片的预览语法可用于检查
DataLab 兼容、Python 风格或 Mathematica 风格的显示效果，但该选择仅影响预览，不会改变计算。参数
表仍由当前公式自动识别，也可以手动增删；常数表关闭时不会参与拟合。

自洽隐式模型用于类似 `u = g(x, u, 参数)`、`y = f(x, u, 参数)` 的问题。每个数据点先求自洽变量，
再把输出表达式代入拟合目标。建议先用较简单的初值和边界确认模型能稳定求解，再增加参数数量或提高
多精度位数。

## 图像与对数坐标

启用绘图后，结果区可显示：

- 拟合曲线与数据点
- 残差图

拟合图支持对数坐标：

- 当数据不满足对数坐标条件时，对数坐标会自动取消，并在日志提示原因

## 输出与导出

- 拟合参数与评估信息会在结果区展示
- 可导出结果表格，并生成用于排版的结果文本与文档预览
