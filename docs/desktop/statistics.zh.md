# 统计平均

统计平均模块用于对一列数据进行统计与均值估计，并可在存在不确定度信息时进行加权统计。

## 输入与模式

- 选择需要统计的数值列
- 描述统计模式会输出计数、均值、可选修剪均值、标准误差、标准差、方差、最小/最大值、中位数、Q1/Q3、IQR、MAD、偏度和超额峰度
- 如存在不确定度信息，可进行加权平均等统计（默认权重为 `w=1/σ²`）
- 对缺失或不合法的不确定度数据行会自动忽略，并在日志给出提示
  - 若存在 `σ=0`：视为“无限权重锚点”；若多个 `σ=0` 数值不一致会报错

## 协方差 / 相关矩阵

统计模块的工作流选择器可对两个或更多显式选择的数值列计算协方差矩阵和
相关矩阵。数值列可写成逗号分隔形式，例如 `A, B, C`。

- 默认使用 listwise 删除：只有所有选中列都有有限数值的行才会参与计算
- 可选 pairwise 删除：每一列对独立选择共同有效行；该结果只作为诊断上下文，
  不作为后续预算汇总候选
- 样本/总体复选框控制协方差分母（`n-1` 或 `n`）
- 加权协方差首版暂缓；现有 sigma 列表达的是测量不确定度，不是已经审阅的
  多变量行权重模型
- CSV 使用长表格式（`matrix`、`row_column`、`column`、`value`、`count`、
  `denominator`）；LaTeX 会导出协方差和相关矩阵表；启用图片时，完整有限的
  相关矩阵会生成热图

## 分组统计

统计模块的工作流选择器可按一个文本列分组，并对每个分组和每个选中的数值列
复用现有的一列统计计算。这样分组统计会继续使用普通统计相同的精度、加权、
不确定度、CSV、LaTeX、图片、工作区、历史记录和报告包路径。

- 分组列是必填的标签列；空白分组标签会被排除并生成诊断
- 数值列可写成逗号分隔形式，例如 `Signal, Reference`
- 支持括号不确定度；显式 sigma 列会覆盖所有选中数值列中的内嵌不确定度
- 首个可见版本保留输入中首次出现的分组顺序，并输出按分组、列、指标、数值
  和不确定度标识的长表
- 在不确定度预算面板中，分组结果只作为诊断上下文，不作为独立物理方差贡献

## Bootstrap 置信区间

统计模块的工作流选择器可运行 Bootstrap 置信区间。首版采用百分位
Bootstrap，并固定为 95% 区间（`2.5%`、`50%`、`97.5%`），以复用现有
Monte Carlo 分布摘要和绘图接口。

- 支持目标统计量：均值、中位数、修剪均值、标准差和方差
- 可选随机种子；设置后串行和并行执行应得到相同的重采样序列摘要
- 重采样次数由核心计算器限制；示例可使用 100 次，正式分析建议使用更大的次数
- Bootstrap 结果会随工作区、历史记录、CSV、LaTeX、图片和报告包保存
- 在不确定度预算面板中，Bootstrap 区间只作为诊断上下文显示，不作为物理方差贡献参与汇总

## 假设检验

统计模块的工作流选择器也可基于同一份内嵌表格数据运行首版假设检验。
桌面端首版包含：

- 单样本 t 检验
- 配对 t 检验（使用 `A - B`）
- Welch 双样本 t 检验
- 精确符号检验
- 卡方拟合优度检验

零假设参数、备择假设、显著性水平、第二列、期望计数或期望概率来源、
拟合参数个数都会随工作区保存。工作区、历史记录、CSV、LaTeX 和报告包
渲染时，会从结构化假设检验 payload 重新生成显示内容。p 值与是否拒绝
零假设只作为诊断上下文显示，不作为不确定度预算中的方差贡献。

## 时间序列 / 滚动统计

统计模块的工作流选择器可直接运行有序序列分析，不需要切换到另一个模块。
首版保持输入行顺序并使用“按行数”的窗口；不会按时间/索引列自动排序，也不
做按时间宽度的重采样。

- 滚动方法：均值、中位数、标准差
- EWMA 平滑需要 `alpha` 或 `span` 二选一；`span` 会按
  `alpha = 2 / (span + 1)` 换算
- 滚动窗口可右对齐或居中；`min_periods` 决定边界点是输出结果还是标记为
  insufficient-window 诊断
- 时间/索引列仅作为标签和单调性诊断，不改变计算顺序
- 仅滚动均值在明确选择对齐的 sigma 列时传播独立输入不确定度
- 时间序列结果会随工作区、历史记录、CSV、LaTeX、图片和报告包保存；在不
  确定度预算面板中，它们只作为诊断序列上下文，不作为方差贡献

## 输出内容

结果区会根据所选模式给出：

- 平均值与对应的不确定度估计
- 95% 均值置信区间。非加权模式使用基于样本标准差的 Student-t 区间，包括总体显示模式；加权模式在没有 `σ=0` 锚点时使用已知 σ 的正态区间
- 标准差、最小值、最大值等统计量
- 描述统计的分位数使用 Hyndman-Fan type 7 插值；样本方差/偏度/峰度或零方差导致不可用时会给出警告诊断
- 可选修剪均值仅用于描述统计：先排序有限值，每端删除 `floor(n * 修剪比例)` 个值，再对剩余值取算术平均。留空或 `0` 表示关闭；无效或过大的比例由核心计算拒绝
- 加权有效样本数 `n_eff`（Kish 公式）等附加信息（如适用）

## 导出与排版

- 可导出结果表格
- 可生成用于排版的结果文本与文档预览
- 内置“统计矩阵：协方差与相关系数”示例展示了带内嵌数据的 listwise
  矩阵计算工作区
- 内置“分组统计：多组多列均值”示例展示了对两个内嵌数值列进行分组加权统计
- 内置“统计平均：Bootstrap 置信区间”示例展示了带固定随机种子的内嵌数据工作区
- 内置“统计检验：单样本 t 检验”示例展示了带内嵌数据的单样本假设检验工作区
- 内置“时间序列：滚动均值”和“时间序列：EWMA 平滑”示例展示了按行数滚动、
  EWMA 平滑和可直接计算的内嵌数据工作区
