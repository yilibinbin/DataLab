# 开发路线图

本路线图仅面向桌面程序，不包含服务端相关内容。

## 已具备能力

- 四个主要模块：序列外推、误差传递、拟合、统计平均
- 结果区展示：结果摘要、运行日志、排版源文件与文档预览
- 表格与图片导出，以及可选的文档编译预览
- 拟合图对数坐标与数据有效性保护
- 离线帮助文档，可从菜单中的帮助入口打开

## 近期计划

- 文档与界面同步：参数说明与界面字段保持一致
- 导出体验优化：导出内容与显示格式更一致，导出文件命名更清晰
- 结果可读性提升：结果摘要结构更清晰，警告信息更聚合

## 中期计划

- 图像交互增强：缩放、导出图片、更多样式选项
- 批处理能力：多批次数据的批量运行与汇总导出
- 可重复性：保存与加载一次运行的完整配置（输入、参数、显示格式）

## 长期计划

- 打包稳定性提升：资源打包与字体一致性更可靠
- 更细粒度的日志等级与错误诊断提示
- 更多模型与方法的界面支持，完善输入输出与说明
