# Roadmap (Desktop)

This roadmap is **desktop-only**. It does not cover web routes or web deployment.

## Current Capabilities

- Four modules: Extrapolation / Error propagation / Fitting / Statistics
- Result area: Values / Log / LaTeX / PDF Preview
- CSV export, LaTeX generation, optional PDF compilation
- `log-x` / `log-y` protection for non-positive data in fitting plots
- Offline documentation via the Help menu

## Near-Term

- Keep docs and UI in sync: help texts, parameter descriptions, and UI fields
- Export UX improvements: clearer filenames and tighter consistency with display formatting
- Improve result readability: better-structured summaries and consolidated warnings

## Mid-Term

- Plot interactions: zooming, image export, more styling options
- Batch workflows: run multiple datasets and export merged reports
- Reproducibility: save/load a full run configuration (inputs, parameters, display settings)

## Long-Term

- Cross-platform packaging stability (resource bundling and font consistency on Windows/macOS/Linux)
- More granular logging levels and better diagnostics
- Extend UI coverage for more models/methods without changing the core numeric algorithms
