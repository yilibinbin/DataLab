# 误差传递

误差传递模块用于对带不确定度的数据执行公式计算，并自动完成误差合成与结果输出。

## 输入内容

- 数据表：首行为表头，后续为数据行
- 常数（可选）：用于参与计算的固定量
- 公式：用界面支持的写法描述计算关系

## 方法选择（Taylor / Monte Carlo）

误差传递支持两类方法：

- **Taylor（偏导）**：快速近似
  - **1 阶**：线性化误差传递（默认）
  - **2 阶**：加入 Hessian（二阶偏导）贡献，并对结果值做均值修正（更接近 Monte Carlo 的均值）
- **Monte Carlo**：按输入的不确定度对每个变量做独立正态抽样，输出“样本均值 ± 标准差”
  - 可设置样本数（至少 100）与随机种子（可复现）

## 公式与函数

程序支持常用的算术运算与函数。为避免写法不一致导致解析失败：

- 建议优先使用界面提供的函数提示与示例
- 变量名称应与表头或常数名称保持一致
- 注意括号配对与运算符优先级

## 输出结果

计算完成后会给出：

- 每行的结果值与合成不确定度
- 可选的不确定度贡献可视化
- 可用于排版的结果文本与导出文件

提示：

- 不确定度贡献分解仅在 Taylor 模式下提供（Monte Carlo 不输出逐变量贡献）。
