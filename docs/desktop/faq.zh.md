# 常见问题

## 一、为什么文档预览生成失败

请按以下顺序排查：

- 确认本机已准备好排版环境，并能正常调用
- 查看运行日志中的详细错误信息（通常比弹窗更完整）
- 确认临时目录可写，生成过程需要创建中间文件

## 二、为什么对数坐标没有生效

当数据不满足对数坐标条件时，对数坐标会自动取消，并在日志提示原因。请检查数据范围是否满足要求。

## 三、为什么导出结果与屏幕显示不一致

导出遵循当前显示格式设置。导出前请先调整显示格式，确保结果区显示满足需求。

## 四、为什么误差传递公式提示解析失败

请检查：

- 是否使用了界面支持的函数写法与运算符
- 变量名称是否与表头或常数名称一致
- 括号是否配对，表达式结构是否完整
