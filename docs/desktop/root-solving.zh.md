# 求根

求根模块用于求解 `F(...)=0` 形式的方程。方程输入框每行一个方程；标量、扫描多根、多项式和方程组模式共用同一套安全表达式、常数代入、精度位数和不确定度解析逻辑。

## 输入

- 方程：填写残差表达式，例如 `x^2 - A`。程序按表达式等于零求根。
- 未知量：填写未知量名称、初始值以及可选上下界。扫描多根模式建议填写上下界。
- 输入数据：有数据列时会按行批量求根；没有数据列时执行单个问题。
- 常数设置：默认关闭。启用后常数会进入公式作用域，但不会作为未知量。

## 求解模式

- 标量：一个未知量，求一个实根。
- 扫描多根：在区间内扫描并返回多个根。
- 多项式：对一元多项式返回全部根。
- 方程组：多个方程和多个未知量联立求解。

## 不确定度

输入值可使用 `1.23(4)`、`3.2898419602500(36)[+9]` 等记号。Taylor 传播复用误差传递的导数思路；Monte Carlo 会对带不确定度输入抽样并重复求根。结果区和 LaTeX 会按“值(不确定度)”格式显示，位数由全局“不确定度位数”控制。

## 绘图

启用“生成图片”时，标量根会绘制残差曲线和根位置；根不确定度过小时会自动添加局部放大图，但主图仍保持真实尺度。二维方程组会绘制两条零等高线和交点；更高维方程组不生成图，只给出提示。
