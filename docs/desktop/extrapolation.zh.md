# 序列外推

序列外推模块用于对多列序列数据进行外推与序列加速，给出外推值与不确定度估计，并支持导出与排版。

## 选择方法

桌面端提供多种外推与加速方法。选择方法后：

- 左侧会显示该方法对应的参数区
- 你可以根据数据规模与收敛情况调整参数，再运行计算

## 常用设置

不同方法的参数不尽相同，常见设置包括：

- 计算精度：高精度计算通常更稳健，但耗时也会增加
- 不确定度显示位数：用于控制结果中不确定度的显示精细程度
- 参考列：用于辅助不确定度估计或对比不同列的收敛表现
- 最大差异列：自动选择与外推值差异最大的参考列

## 结果与输出

计算完成后，结果区通常会展示：

- 外推值、不确定度与提示信息
- 可选的趋势图或对比图
- 可用于排版的结果文本与导出文件
