# 导出与排版

本页介绍桌面端的导出与排版相关功能，以及数值显示格式的设置方式。

## 表格导出

计算完成后可导出结果表格。导出的数值显示规则与结果区保持一致：

- 未启用科学计数法时，按设置的小数位数进行显示与四舍五入
- 启用科学计数法时，按设置的有效位数进行显示与四舍五入

建议在导出前先调整好显示格式，以便导出内容与屏幕显示一致。

## 排版源文件

勾选生成排版源文件后，程序会输出适合整理成表格或报告的排版文本：

- 结果通常会采用括号记号表示不确定度，便于阅读与排版
- 你可以在对应的标签页中查看与编辑生成的排版文本

## 文档编译与预览

如需生成文档预览，需要本机已安装并配置好相应的排版环境。若编译失败：

- 优先查看运行日志中的错误信息
- 确认临时目录可写，编译过程需要创建中间文件

## 表格对齐

部分排版模板支持更精细的数值列对齐。若启用对齐后出现编译错误，可尝试关闭对齐选项，或在模板中补齐所需设置。
