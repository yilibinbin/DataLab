# 桌面端帮助文档

本目录为桌面程序的离线帮助文档，用于介绍界面流程、主要功能与导出排版。

## 主要功能

桌面端包含四个主要模块：

- 序列外推：对多列序列数据进行外推与序列加速，给出外推值与不确定度估计
- 误差传递：对带不确定度的数据按公式计算，并完成误差合成
- 拟合：对数据进行显式模型拟合，输出参数、不确定度与拟合优度信息
- 统计平均：对数据进行统计与均值估计，支持加权与非加权

通用能力包括：

- 结果区集中展示：结果摘要、运行日志、排版源文件与文档预览
- 表格与图片导出，便于整理与汇报
- 拟合图支持对数坐标，并对不满足条件的数据自动保护

## 建议阅读顺序

- 先阅读《使用指南》，了解界面布局与基本流程
- 若希望了解算法细节与输出字段含义，阅读《理论说明》
- 根据需求进入对应模块文档：序列外推、误差传递、拟合、统计平均
- 涉及导出与排版时，阅读《导出与排版》
