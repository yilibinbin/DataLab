# 拟合功能

对数据进行曲线拟合，获取最佳拟合参数。

## 使用步骤

1. **输入数据**：上传或粘贴 `x y` 格式数据
   ```
   x y
   1.0  2.1(5)
   2.0  4.2(5)
   3.0  6.0(5)
   ```

2. **选择拟合模式**：
   - **自动选择**：系统自动尝试多个模型
   - **预设模型**：从模型库选择
   - **多项式拟合**：指定阶数
   - **反幂级数**：指定幂次范围
   - **自定义模型**：输入自定义表达式

3. **设置选项**：
   - 加权拟合：使用不确定度加权
   - 对数坐标：选择 log-x, log-y, 或 log-log

4. **查看结果**：
   - 拟合参数及误差
   - 质量指标（χ², AIC, BIC, R², RMSE）
   - 拟合曲线与残差图

## 支持的模型

- 多项式拟合
- 反幂级数拟合
- Padé 近似
- 预设模型库（对数、指数组合等）
- 自定义模型
