# 常见问题

## 使用问题

### 公式报错

**Q: 为什么我的公式报错？**

A: 请检查：
- 函数使用 Mathematica 风格首字母大写：`Sin[x]` 而非 `sin(x)`
- 列名或常数名拼写正确
- 使用了支持的函数（Sin, Cos, Log, Exp, Sqrt, Abs）

### 列名引用

**Q: 如何在公式中引用列？**

A: 支持三种方式：
- 直接使用表头列名（如 `E1`, `E2`）
- 使用别名 `x1`, `x2`, `x3` ... 按列顺序
- 外推模式下使用 `A`, `B`, `C` 表示前三列

### 数据格式

**Q: 不确定度如何表示？**

A: 支持两种格式：
- 括号记号：`1.2345(67)` 表示 1.2345 ± 0.0067
- 独立列：值和不确定度分两列

### 对数坐标

**Q: 对数坐标拟合有什么限制？**

A:
- `log-x` 要求所有 x > 0
- `log-y` 要求所有 y > 0
- `log-log` 要求所有 x, y > 0

## 技术问题

### LaTeX 编译失败

**Q: PDF 生成失败怎么办？**

A:
- 确保系统已安装 LaTeX（texlive 或 mactex）
- 检查 LaTeX 引擎路径设置
- 查看错误提示中的详细信息

### 性能问题

**Q: 高精度计算很慢？**

A: 这是正常现象。`mp.dps` 越高计算越慢。建议：
- 仅在需要时使用高精度
- 幂律/序列加速默认 80 位已足够
- 一般计算使用默认 16 位

### 文件上传

**Q: 文件上传失败？**

A: 检查：
- 文件是否为 UTF-8 编码
- 文件大小是否超过限制（默认 1MB）
- 文件格式是否正确（首行表头，后续行数据）

## 故障排查

### 计算结果异常

1. 检查输入数据格式
2. 验证公式语法
3. 确认参数设置合理
4. 查看警告信息

### 页面显示问题

1. 清除浏览器缓存
2. 尝试其他浏览器
3. 检查 JavaScript 控制台错误

### 导出功能问题

**CSV 下载**：
- 确保结果已生成
- 检查浏览器下载设置

**LaTeX/PDF**：
- 复制 LaTeX 代码手动编译
- 检查 PDF 查看器设置
