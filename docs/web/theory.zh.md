# DataLab 理论说明（模型、算法与输出字段）

本文档基于 DataLab 当前源码实现，目的是说明：各功能模块背后的数学模型、默认假设、以及“计算结果包含哪些内容”。若界面文案与本实现不一致，以代码行为为准。

---

## 1. 统一数据表示

### 1.1 多精度数值

DataLab 的核心计算统一使用 `mpmath` 的 `mp.mpf`（多精度浮点）。多数计算入口会临时设置 `mp.dps`（十进制有效位数），以兼顾高精度与可重复性。

### 1.2 不确定度（σ）的两种输入方式

1) **括号记号**：`value(digits)[exp]`

- `1.2345(67)` 表示 `1.2345 ± 0.0067`
- `1.2345(67)[-2]` 表示 `(1.2345 ± 0.0067) × 10^-2`

2) **独立 σ 列**：在数据表中单独提供一列 `sigma`（或界面指定的 σ 列）。

### 1.3 列名规范化与别名（x1/x2/A/B/C）

公式系统会把表头列名规范化为安全标识符（将非字母数字替换为 `_`，首字符为数字则加前缀），并在必要时用 `_2/_3...` 去重。

- 误差传递：支持旧式 `x1/x2/...`，会在计算前替换为规范化列名（并可产生提示）。
- 外推自定义公式：支持列名、`x1/x2/...`，并额外支持 `A/B/C` 指代前三列（兼容三点外推习惯）。

## 2. 统一“安全公式语言”（外推自定义公式 / 误差传递 / 拟合自定义模型）

三处自定义公式**共享同一套解析与白名单**：`data_extrapolation_latex_latest.py:safe_eval`。

### 2.1 语法与安全规则（AST 白名单）

- 支持：数字、变量名、括号、`+ - * / ** %`、一元 `+/-`、函数调用（仅限白名单）。
- 支持 Mathematica 风格：`Sin[x]`（中括号会自动转圆括号）、`^`（自动转 `**`）。
- 禁止：属性访问（如 `a.__class__`）、关键字参数、任意对象/字符串常量、以及所有不在白名单中的语法节点。
- 错误风格：抛出与外推/误差传递一致的 `ValueError`，并保持一致的中文/英文双语消息结构。

### 2.2 支持的常数与函数（同一份白名单）

常数（区分大小写）：

- `Pi`, `E`

函数（需首字母大写）：

- 基本：`Sin`, `Cos`, `Tan`, `Asin`, `Acos`, `Atan`, `Sinh`, `Cosh`, `Tanh`, `Asinh`, `Acosh`, `Atanh`
- 指数/对数：`Exp`, `Log`, `Ln`, `Log10`, `Sqrt`, `Power`
- 常用：`Abs`, `Erf`, `Gamma`
- 特殊函数：`Zeta`, `Hyp0f1`, `Hyp1f1`, `Hyp2f1`, `PolyLog`, `BesselJ`, `BesselY`, `Airy`

## 3. 序列外推（Extrapolation）

外推模块对“多列序列数据”逐行计算外推值 `V` 与不确定度估计 `U`，并返回每行的元信息（method/details）。

### 3.1 默认三点公式（quadratic）

对三列数据 `A,B,C`：

- 外推值：`V = ((C - B)^2) / (B - A) + C`
- 不确定度（保守差异上界）：`U = max(|V-A|, |V-B|, |V-C|)`

当 `B≈A` 时避免除零：退化取 `V=C`，并仍用上式差异定义 `U`。

### 3.2 幂律外推（power_law）

幂律模型假设：

- `E(x) = E_inf + a * x^(-p)`

输入：三点能量 `(E1,E2,E3)` 与三点基数 `(x1,x2,x3)`。

实现步骤（高精度）：

- 计算比值 `R = (E1-E2)/(E2-E3)`
- 解方程：`(x1^{-p}-x2^{-p})/(x2^{-p}-x3^{-p}) = R` 得指数 `p`
- 求振幅：`a = (E1-E2)/(x1^{-p}-x2^{-p})`
- 外推：`E_inf = E1 - a*x1^{-p}`

### 3.3 序列加速（Richardson / Shanks / Wynn ε / Levin u）

调用 `mpmath` 的序列加速实现：

- Richardson：`mp.richardson`
- Shanks / Wynn ε：`mp.shanks`（其本质为 Wynn ε 算法；仅标签不同）
- Levin：`mp.levin(variant='u'/'t'/'v')`

这些方法会产生 `metadata`（例如 `error_estimate`、`cancellation_weight` 等）供展示/诊断。

### 3.4 外推不确定度 `U` 的定义：参考列差异

除 “默认三点公式” 外，幂律外推/序列加速/自定义公式的 `U` 统一取：

- `U = |V - reference|`

其中 `reference` 来自“参考列”（默认第三列），也可由用户指定；并支持 `auto_max_diff` 自动选择“与 V 差异最大的列”作为参考列，以获得更保守的 `U`。

### 3.5 外推输出字段

逐行输出（核心字段）：

- `value`：外推值 `V`
- `uncertainty`：外推不确定度 `U`
- `method`：`quadratic/power_law/richardson/shanks/wynn_epsilon/levin_u/custom`
- `details`：方法相关信息（例如 `reference_column`、幂律的 `exponent/amplitude`、加速器 `metadata`、自定义 `formula`）

## 4. 误差传递（Error Propagation）

误差传递模块对每一行带不确定度的数据，根据公式 `f(x)` 计算输出值与输出不确定度。

### 4.1 一阶误差传播公式

对独立输入量 `x_i`（标准不确定度 `σ_i`），输出 `f` 的一阶传播为：

- `σ_f = sqrt( Σ ( ∂f/∂x_i * σ_i )^2 )`

实现中只对 `σ_i > 0` 的变量参与合成（σ=0 视为无贡献）。

二阶（`order=2`）会在一阶项基础上加入 Hessian 的二阶贡献，并对输出值做均值修正（更接近 Monte Carlo 的输出均值）：

- `E[f] ≈ f(μ) + 1/2 * Σ_i (∂²f/∂x_i²) * σ_i²`
- `Var[f] ≈ Σ_i (∂f/∂x_i)² σ_i² + 1/2 * tr((H Σ)²)`

其中 `H` 为 Hessian，`Σ` 为输入协方差矩阵；当前实现默认 `Σ` 为对角阵（独立假设）。

### 4.2 偏导数获取：符号优先，数值差分兜底

实现优先尝试符号微分（Sympy），若失败则回退到数值差分。

数值偏导使用中心差分：

- `∂f/∂x ≈ (f(x+h) - f(x-h)) / (2h)`

步长 `h` 会随 `mp.dps` 自适应（经验上：一阶 `h ~ eps^(1/3)`，二阶 `h ~ eps^(1/4)`，并按 `max(1,|x|)` 缩放），以平衡截断误差与舍入误差。

### 4.3 只对“公式实际引用”的输入做传播

程序会对公式做 AST 扫描，识别实际引用的列/常数，只对这些输入计算偏导与贡献；若公式无法解析，则退回“对所有输入传播”的旧行为（保证兼容）。

### 4.4 贡献分解（variance contributions）

可选输出每个输入对总方差的贡献（Taylor 模式）：

- `contrib_i = ( ∂f/∂x_i * σ_i )^2`
- 总方差：`σ_f^2 = Σ contrib_i`

该贡献映射会按显示名（列名/常数名）聚合。

### 4.5 误差传递输出字段

逐行输出（核心字段）：

- `value`：
  - Taylor `order=1`：`f(μ)`（工作点处的公式值）
  - Taylor `order=2`：`≈ E[f]`（含均值修正）
  - Monte Carlo：样本均值
- `uncertainty`：输出的标准不确定度（标准差）
- `contributions`（可选）：每个输入的方差贡献（用于贡献柱状图/诊断）

## 5. 拟合（Fitting）

拟合模块对数据 `(x_i, y_i)` 寻找模型参数 `p`，并输出参数估计、不确定度、协方差矩阵与拟合优度指标。

### 5.1 目标函数 χ² 与权重

残差：`r_i = y_model(x_i; p) - y_i`

- **非加权**：`χ² = Σ r_i^2`
- **加权**：`χ² = Σ w_i * r_i^2`，典型 `w_i = 1/σ_i^2`

`dof = n - k`，其中 `n` 为数据点数，`k` 为自由参数数目（线性模型为系数数；自定义模型为 free params 数）。

### 5.2 显式线性模型（多项式 / 1/x^p）

线性模型形如：

- `y ≈ Σ b_j φ_j(x)`（对参数线性）

实现采用 QR 分解解最小二乘，并在加权情况下把设计矩阵按 `sqrt(w_i)` 缩放（等价于最小化加权 χ²）。

当前 Web 拟合界面公开的线性基函数包括多项式与 `1/x^p` 展开；Padé 与幂律极限模板在后续显式模型步骤中处理。

### 5.3 非线性/自定义模型（含模板：power_limit、Padé）

自定义模型表达式通过 `fitting/model_parser.py:build_model_specification` 构建：

- 求值：仍使用统一的 `safe_eval`（与外推/误差传递同源）
- 参数偏导：对每个参数用同一套数值偏导（中心差分）实现
- 求解：用 `mp.findroot` 解 “梯度=0”（即 `∂χ²/∂p_j = 0` 的方程组），并自动尝试多个初值扰动以降低局部失败概率

模板模型：

- **幂律极限模型**：`A*x**(-p) + C`（约束 `p ≥ 0.1`）
- **Padé(m|n)**：`(a0 + a1 x + ... + a_m x^m) / (1 + b1 x + ... + b_n x^n)`

### 5.4 协方差矩阵是什么？如何计算？

在最优解附近，把模型线性化，参数协方差近似为：

- 构造雅可比矩阵 `J_{i,j} = ∂y_model(x_i; p)/∂p_j`（在最优参数处）
- 若使用权重，则实现等价于使用 `sqrt(w_i)` 对 `J` 的每一行做缩放（即 `J ← diag(sqrt(w)) J`）
- 计算 `Cov ≈ (J^T J)^{-1} * σ²`
- 其中 `σ²` 由残差估计：`σ² ≈ χ² / dof`

参数标准不确定度（统计分量）来自对角线：

- `σ(p_j) = sqrt(Cov_{j,j})`

若 `J^T J` 不可逆（奇异/病态），实现会给出告警，并把相关不确定度置为 `NaN`（不可用）。

### 5.5 统计误差 vs 系统误差（DataLab 的实现口径）

FitResult 中把参数误差拆为：

- `param_errors_stat`：由协方差得到的统计误差（χ² 近似下的标准差）
- `param_errors_sys`：当提供了数据 σ 且**未作为权重使用**时，程序会把所有 `y_i` 同时做 `±σ_i` 扰动并重新拟合，取参数偏移的平均值作为系统误差估计
- `param_errors_total`：总误差 `sqrt(stat^2 + sys^2)`（二次和）

注意：核心拟合引擎支持 “σ 作为权重”（统计加权）与 “σ 作为系统扰动”（±σ 重拟合）两种口径；不同前端可能会选择其一。

实现差异（按当前源码）：

- 桌面端：勾选“统计误差加权”会构造 `w=1/σ²` 进入 χ² 与协方差；为避免双计，不再额外计算系统误差。
- Web 端：当前 `fit_weighted` 会把 σ 作为 `data_sigmas` 用于 `y±σ` 重拟合（系统误差估计），但不做 χ² 加权。

### 5.6 依赖参数（expr）与误差传播

参数支持：

- `fixed`：固定值
- `min/max`：边界
- `expr`：依赖其它参数的表达式（用受限的 Sympy 环境解析）

当存在 `expr` 参数时，程序会用链式法则把自由参数的协方差传播到依赖参数：

- `Var(g(p)) ≈ J_g Cov J_g^T`

### 5.7 拟合输出字段（FitResult）

核心输出包含：

- `params`：最优参数字典
- `param_errors_stat/sys/total`：误差分量与总误差（`param_errors` 为兼容字段，等价 total）
- `covariance`：协方差矩阵（按自由参数顺序；依赖参数误差通过传播得到）
- `residuals`：每个点的残差 `r_i`
- `fitted_curve`：每个点的拟合值 `y_model(x_i)`
- `chi2`, `reduced_chi2`, `aic`, `bic`, `r2`, `rmse`
- `details`：表达式、是否加权、边界命中/协方差告警等信息

## 6. 统计平均（Statistics）

统计模块对一列数据（可选 σ）计算均值与不确定度摘要。

### 6.1 算术平均

- `mean = Σ x / n`
- 标准差：样本/总体两种口径（分母分别为 `n-1` 或 `n`）
- 均值标准误差：`σ_mean = std / sqrt(n)`

### 6.2 加权平均（1/σ²）

当提供 σ 且选择加权模式：

- 权重 `w_i = 1/σ_i^2`
- `mean = (Σ w_i x_i) / (Σ w_i)`
- `σ_mean = sqrt(1 / Σ w_i)`
- 可选输出有效点数：`n_eff = (Σ w)^2 / Σ(w^2)`

特殊处理：若存在 `σ=0` 的点，视为“无限权重锚点”；若多个 `σ=0` 点数值不一致则报错。

### 6.3 统计输出字段

- `mean`：均值
- `std_mean`：均值标准误差
- `std`：标准差
- `v_min/v_max`：最小/最大值
- `effective_n`：加权有效点数（可选）
- `dropped`：因缺失/非法 σ 被忽略的行数

## 7. 导出与排版（LaTeX / PDF / CSV）

所有模块均支持把结果格式化为“括号不确定度 + 指数括号”形式，并生成：

- `CSV`：用于进一步处理
- `LaTeX`：表格（可选 `dcolumn`/`siunitx` 对齐、数字分组）
- `PDF`：在存在 LaTeX 环境时可安全编译预览（Web 端可选）
