## 功能概览

DataLab Web 是一个基于浏览器的高精度数值计算工具，提供以下四大功能模块：

### 序列外推
对数值序列进行外推，获得极限值或趋势预测。支持多种外推方法：
- 幂律外推（三点法）
- Richardson 序列加速
- Shanks 变换 / Wynn ε 算法
- Levin u-transform
- 自定义公式外推

### 误差传递
计算带不确定度数据通过公式后的误差传递。特性包括：
- 支持数值偏导计算
- 自动合成不确定度
- 支持常数与数据的联合传播
- 可视化不确定度贡献分解

### 拟合
对数据进行曲线拟合，获取最佳拟合参数。支持：
- 多项式拟合
- 反幂级数拟合
- Padé 近似
- 预设模型库
- 自定义模型

### 统计平均
对带不确定度的数据进行加权或非加权统计平均，支持多种统计模式。

## 数据输入格式

DataLab Web 支持两种不确定度表示方法：

**括号记号**：`值(末位不确定度)[10的指数]`
```
1.2345(67)       # 表示 1.2345 ± 0.0067
1.2345(67)[-2]   # 表示 1.2345 ± 0.0067 × 10^-2
```

**独立列**：值和不确定度分两列
```
值       不确定度
1.2345   0.0067
2.3456   0.0089
```

## 输出格式

- **CSV 下载**：可导出表格数据
- **LaTeX 格式**：自动生成带括号记号的 LaTeX 表格代码
- **PDF 导出**：支持直接编译 PDF（需要 LaTeX 环境）
- **可视化图表**：外推趋势图、误差贡献图等

## 开始使用

1. 在主页选择功能模块（外推/误差传递/拟合/统计）
2. 输入数据（粘贴文本或上传文件）
3. 配置参数和选项
4. 点击运行按钮查看结果
5. 下载 CSV、复制 LaTeX 或导出 PDF
