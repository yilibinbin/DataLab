# 导出与排版（Web）

本页介绍 Web 端结果的导出与排版选项，包括 CSV、LaTeX、PDF 以及数值显示格式。

## CSV 下载

各模式计算完成后，结果表下方提供“下载 CSV”按钮：

- CSV 的数值列与当前页面的显示格式一致：
  - 未勾选“使用科学计数法”时：按“小数位数”四舍五入
  - 勾选“使用科学计数法”时：按“有效位数”四舍五入并以科学计数法显示
- LaTeX 列会按页面展示的 LaTeX 文本导出（用于粘贴到论文或进一步排版）。

## LaTeX 输出

外推、误差传递、拟合、统计均支持生成 LaTeX 表格文本：

- LaTeX 列使用括号记号（例如 `1.2345(67)`），并遵循“结果不确定度有效位数”的设置。
- 可选开启 `dcolumn` 对齐：用于在 LaTeX 表格中按小数点对齐数值列。

## PDF 编译

可勾选“尝试编译 PDF 并提供下载”：

- 需要系统已安装 LaTeX 引擎（如 `pdflatex` 或 `xelatex`）。
- 可在“LaTeX 引擎（可选）”里填写可执行文件路径；留空则使用系统默认搜索路径。

若 PDF 编译失败：

- 请先确认 LaTeX 引擎可在命令行运行；
- 若是路径/权限问题，建议在服务器上为运行账户配置正确的可执行权限与临时目录权限。

## `dcolumn` 的使用建议

- 当你的论文模板已启用 `dcolumn` 或 `siunitx`，并希望表格对齐更美观时再开启。
- 若你的模板未包含相关宏包，开启后可能导致编译失败；此时请取消勾选或在模板中补齐宏包。

## 显示格式：小数位数 vs 有效位数

结果表右上角提供显示控制：

- 未勾选“使用科学计数法”时：输入框表示“小数位数”
- 勾选后：输入框表示“有效位数”

该规则仅影响页面显示与 CSV 导出，不影响 LaTeX 列的格式化规则。
