# 序列外推

序列外推用于对数值序列进行外推，获得极限值或趋势预测。

## 使用步骤

1. **输入数据**：粘贴或上传多列数据
2. **选择外推方法**（见下文）
3. **设置参数**：
   - 参考列：用于计算不确定度
   - 最大差异列：自动选择与外推值差异最大的参考列（更保守）
   - 多精度 mp.dps：数值计算精度
   - 结果不确定度有效位数
4. **生成输出**：点击“运行外推并生成 LaTeX”

## 外推方法

### 幂律外推（三点法）
适用于 `f(x) = A*x^(-p) + C` 形式的序列。

**参数**：
- x1, x2, x3：三个点的 x 坐标
- power_exponent：固定幂指数 p（可选）
- power_seed：p 的初始值

**适用场景**：量子化学基组外推、数值积分精度外推

### Richardson 序列加速
适用于渐近展开序列。

**适用场景**：数值微分、积分、有限差分方法

### Shanks 变换 / Wynn ε 算法
通用序列加速算法。

**适用场景**：级数求和、迭代法加速

### Levin u-transform
专门用于加速震荡或交替级数。

**参数**：
- levin_variant：Levin 变体（默认 'u'）

### 自定义公式
使用自定义数学公式进行外推。

**公式语法**：
- 使用 A, B, C 或列名表示数据列
- 使用 x1, x2, x3 作为列的别名
- 支持函数/常数：与“误差传递/拟合自定义模型”共享同一套白名单（以界面“公式帮助”为准；也可参考[理论说明](theory.md)中的完整列表）
- 示例：`(C - B)^2/(B - A) + C`
